# Calculadora em Python

Este projeto é uma calculadora simples desenvolvida em Python, com o objetivo de praticar lógica de programação.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erro (divisão por zero)
- Validação de operação

## Tecnologias utilizadas
- Python
